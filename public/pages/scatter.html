Analisis Scatter Diagram & Regresi Linear

Tema Latar:

📖 Fungsi Chart Ini
Scatter Diagram (Diagram Tebar) digunakan untuk menganalisis dan memetakan hubungan (korelasi) antara dua variabel: Variabel Bebas (X) dan Variabel Terikat (Y). Alat ini sangat berguna untuk memverifikasi apakah ada korelasi sebab-akibat yang nyata dan memodelkan garis regresi linear sebagai dasar prediksi perbaikan proses.

🧮 Rumus Perhitungan

▼
Koefisien Korelasi Pearson (r)
r = 
NΣXY - (ΣX)(ΣY) √ [NΣX² - (ΣX)²] [NΣY² - (ΣY)²]
Pembilang = NΣXY - ΣXΣY
Penyebut = √[NΣX² - (ΣX)²] [NΣY² - (ΣY)²]
Mengukur kekuatan dan arah hubungan linear antara variabel X dan Y. Nilai mendekati 1 atau -1 berarti hubungan erat, mendekati 0 berarti hubungan lemah/acak.
Koefisien Determinasi (R²)
R² = (r)²
Menunjukkan persentase kontribusi pengaruh variabel bebas X terhadap naik-turunnya variabel terikat Y. Sisanya ditentukan faktor lain di luar model.
Persamaan Regresi Linier (ŷ)
ŷ = a + bX
Model matematika garis lurus (trendline) untuk memprediksi nilai variabel Y berdasarkan nilai variabel X.
Kemiringan Garis / Slope (b)
b = 
NΣXY - (ΣX)(ΣY) NΣX² - (ΣX)²
Slope (b) mengukur laju perubahan nilai Y jika variabel X naik sebesar 1 satuan. Menunjukkan arah hubungan tren.
Titik Potong / Intercept (a)
a = 
ΣY - b(ΣX) N
Intercept (a) adalah nilai awal perkiraan Y ketika nilai variabel bebas X sama dengan nol (titik potong garis pada sumbu tegak Y).
Glosarium Hubungan X & Y: Variabel X | Variabel Y | Korelasi (r)
